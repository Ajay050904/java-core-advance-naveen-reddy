### **Q: Explain Data, Data Types, and Data Structures**

**Answer (Concise Points):**

* **Data**

  * Core of all software applications
  * Taken from user, processed, displayed, or stored in databases

* **Variables**

  * Used to store data
  * Can be imagined as a **box** holding data

* **Data Types**

  * Define the type of data a variable can store

  * Used when primitive types are not enough
  * Represent real-world entities (Human, Phone, Student)
  * Examples:

    * `struct` in C
    * `class` in OOP languages

* **Data + Operations**

  * Every data type supports operations
  * Integer â†’ add, subtract, multiply, divide
  * String â†’ length, concatenate, substring

* **Storing Multiple Data**

  * Use **arrays, lists, sets**
  * One structure stores many values
  * Supports operations like insert, delete, search, access
